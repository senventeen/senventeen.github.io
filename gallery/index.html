南京牛首山

照片拍摄地位于南京的牛首山下，这是进山之前拍的。那天带着相机去，可惜的是刚在山脚下相机就没电了。不过还是拍了很多很满意的照片！牛首山绝对是一个很值得去的地方，太壕了！

<

南京洪山森林动物园

照片拍摄地位于南京的红山森林动物园的门口。小谭那天的装扮刚好也是小学生出游装，哈哈哈。就是说感觉很搭，看了大熊猫真的大，然后还在一个地方小睡了一会儿...

<

南京博物院

这个地方是实验室师兄推荐来的，说实话，确实是不虚此行。而且小谭当天的旗袍刚好跟这个风格很搭，看到了很多很有趣的东西，还有镇馆之宝银缕玉衣！

<

南京金陵小城

幸而路金陵，去南京的第一天就去了金陵小城。买的票刚好踩点赶上。里面的夜景真的好好看，有种穿越回古代的感觉。

<

南京金陵小城二

金陵小城处处都是这种风格，而且当天急急忙忙去弄的汉服，效果很不错噢，虽然路上有点赶，但是到了之后就感觉都很值得。

南京玄武湖

感觉去玄武湖的那个下午是很从容的一个午后，正值五一旅游旺季，在湖边歇歇脚。跟喜欢的人一起看日落，真的是一件很幸福的事情。

<

南京数学建模

一下子五一就要结束了，当时真好小谭的数学建模选拔赛，到写下描述的今天国赛都打完了，回头看还是蛮不错的一个经历的呢！
